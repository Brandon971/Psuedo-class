What is a Pseudo-class

A Pseudo-class is a selector that selects elements that are in specific state

For example they are the first element of their type, or they are being hovered over by the mouse pointer

Its also a keyword added to a selector that specifies a special state of the selected elements.for like the psuedo-class :hover can be used to select a button when a users pointer hovers over the button and this selected button can then be styled.

There are 9 different types of pseudo-classes

  • Element display state pseudo-classes
  • Input pseudo-classes
  • Linguistic pseudo-classes
  • Location pseudo-classes
  • Resource state pseudo-classes
  • Time-dimensional pseudo-classes
  • Tree-structural pseudo-classes
  • User action pseudo-classes
  • Functional pseudo-classes
